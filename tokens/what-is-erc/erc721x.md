# ERC721x

ERC721x is an extension of [tokens/what-is-erc/erc721.md](ERC721) that adds support for multi-fungible tokens and batch transfers, while being fully backward-compatible.

ERC721x is heavily inspired by the [ERC1178](https://github.com/ethereum/EIPs/pull/1178/) token standard.

Reference: [https://erc721x.org/](https://erc721x.org/)
